# Knowledge Book

A book that when you right click on it gives you all recipes!

Version: 1.21.X

## Commands

Give:

```mcfunction
/loot give @s loot knowledge_book:knowledge_book
```

Disable:

```mcfunction
/datapack disable "file/knowledge-book.zip"
```

Enable:

```mcfunction
/datapack enable "file/knowledge-book.zip"
```
